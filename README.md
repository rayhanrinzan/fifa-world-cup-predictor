# FIFA World Cup 2026 Match Prediction Project

This is a machine learning project for predicting international football match outcomes and eventually simulating the 2026 FIFA World Cup.

## Current Goal

Build a simple model that predicts whether the home team wins, draws, or loses based on historical international match data.

## Project Roadmap

1. Data exploration
2. Baseline model
3. Feature engineering
4. Neural network
5. Evaluation
6. Tournament simulator
7. FastAPI backend
8. React frontend
9. Deployment
